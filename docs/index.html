3ステップで運用開始

1

インストール

pip install ops-audit-cli

Python標準ライブラリのみ。依存関係ゼロ。

2

ログを用意

運用ログをJSON形式で準備。OpenClaw等からエクスポート可能。

3

監査実行

ops-audit log.json report.json

結果をJSONで出力。CI/CD連携も簡単。
