# Public Release Approval Pack v1

## 目的
`projects/agent-economy` のMVPを対外公開するための**最終承認を1回で判断できる**材料をまとめる。

## 公開対象（最小スコープ）
- `README.md`
- `ops-audit-schema-v1.json`
- `ops-audit-rules-v1.md`
- `ops_audit_cli.py`
- `sample_input.json`
- `sample_output.json`
- `beta-test-report-2026-02-26-v1.md`

## 非公開（今回出さない）
- 実運用ログ（個人情報・環境情報が混ざる可能性があるもの）
- APIキー/認証情報/接続先情報
- マスターの実名・ハンドル・特定可能情報

## 公開前Go/No-Go（承認者チェック）
1. 個人情報が含まれない
2. 認証情報が含まれない
3. 規約違反リスクがない（スクレイピング禁止領域・無断転載なし）
4. 再現手順がREADMEに記載されている
5. 「実験版」であることを明記している

## リリース文面（短文）
AI運用監査MVP（実験版）を公開しました。JSON入力を監査ルールに通し、警告/要修正/安定を機械判定します。サンプル入出力とテスト結果を同梱しています。改善提案歓迎です。

## 受け入れ条件（公開完了の定義）
- GitHubで上記最小スコープが閲覧可能
- READMEから実行再現ができる
- 1本目の告知文（Xまたはnote）が公開済み

## 実行ランブック（30分）
- 0-5分: 最終差分確認（公開対象のみ）
- 5-10分: Go/No-Goチェック
- 10-20分: GitHub公開（タグ任意）
- 20-30分: 告知文1本を公開し、URLを記録

## blocker（現時点）
公開実行は**公開先ポリシーの最終承認1回**が必要。
